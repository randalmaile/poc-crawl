What can Google actually crawl within Angular...
This is a small test to find out.

Project Description

{{projectDescription}}

Project Description is defined in the .ts file and added to the projectDescription variable in the template.

  • Can Google see pages generated by Angular?
  • Can Google see parts of a page dynamically generated?
  • Does Google recognized Meta titles, descriptions, and canonicals?
  • Is Google able to navigate to pages in an Angular app without visible UI paths? (notice we don't have a navigation bar)
  • this.seoService.setTitle('How Google crawls Angular - Home | TIWed Inc.');
  • this.seoService.setMetaDescription('Home of the site that tests how Google crawls Angular - A site by TIWed Inc. that tests the ins and outs of how Google crawls Angular sites.');
  • this.seoService.setCanonicalLink('http://www.tiwedinc.com');
  • this.seoService.setMetaRobots('index');
